# 🚀 Brokos Dev – Portfolio

Nowoczesne portfolio front-end developera stworzone w React + Vite + TailwindCSS.  
Projekt prezentuje moje umiejętności oraz przykładowe realizacje.

## 🖥️ Live

👉 https://broki04.github.io/brokos-dev/

## 📂 Repozytorium

👉 https://github.com/broki04/brokos-dev

---

## ✨ Funkcje

- Responsywny design (desktop-first)
- Dostosowanie do różnych urządzeń (desktop, tablet, mobile)
- Nowoczesny UI (TailwindCSS)
- Komponentowa architektura (React)
- Szybkie ładowanie dzięki Vite
- Przejrzysta struktura kodu

---

## 🛠️ Technologie

- React
- TypeScript
- Vite
- TailwindCSS

---

## 🔮 Plany rozwoju

- dodanie trybu dark/light
- integracja z back-endem
- animacje (biblioteka Framer Motion)

---

## ⚙️ Instalacja i uruchomienie

```bash
git clone https://github.com/broki04/brokos-dev.git
cd brokos-dev
npm install
npm run dev
```
